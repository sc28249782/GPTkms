`pages/` stores compiled knowledge that can be edited and refined.

`raw/` stores immutable evidence or captured notes.

`inbox/` is for uncategorized material waiting for cleanup.

`snapshots/` is reserved for exports or periodic summaries.

For a reusable memory rule, see [Promotion candidate memory pattern](../concepts/promotion-candidate-memory-pattern.md).
